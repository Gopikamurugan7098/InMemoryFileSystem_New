# InMemoryFileSystem_New

## Project Description
This project is a Java-based In-Memory File System that simulates a basic file system using Tree Data Structure. It allows users to create directories and files, navigate through directories, and display the file system hierarchy without using the actual operating system's file storage.

## Features
- Create directories
- Create files
- Organize files using a tree structure
- Navigate between directories
- Display file system hierarchy
- Object-Oriented Design in Java

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Tree Data Structure

## Project Structure
```
src/
  Main.java
  Directory.java
  FileSystem.java
  MyFile.java
  Node.java
```

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Gopikamurugan7098/InMemoryFileSystem_New.git
   ```

2. Open the project in VS Code or any Java IDE.

3. Compile and run:
   ```bash
   javac src/*.java
   java src.Main
   ```
